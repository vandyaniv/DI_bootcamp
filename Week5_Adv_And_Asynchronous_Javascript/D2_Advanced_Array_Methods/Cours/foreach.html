Exercices pour comprendre la méthode forEach() en JavaScript

Voici une série d'exercices qui montent en difficulté pour t'aider à comprendre et maîtriser la méthode forEach(). Chaque exercice est conçu pour te faire appliquer forEach() dans des situations de plus en plus complexes.

Exercice 1 : Afficher chaque élément du tableau

Objectif : Utiliser forEach() pour afficher chaque élément d'un tableau dans la console.

Instructions : Crée un tableau fruits avec les éléments suivants : "pomme", "banane", "orange". Utilise forEach() pour afficher chaque fruit dans la console. 

const fruits = ["pomme", "banane", "orange"];

// Utilise forEach pour afficher chaque fruit
fruits.forEach(function(fruit) {
  console.log(fruit);
});

Exercice 2 : Afficher les éléments avec leur index

Objectif : Utiliser forEach() pour afficher chaque élément d'un tableau avec son index.

Instructions : Utilise le même tableau fruits et modifie ton code pour afficher chaque fruit avec son index, par exemple : 0: pomme, 1: banane, etc.

const fruits = ["pomme", "banane", "orange"];

// Utilise forEach pour afficher chaque fruit avec son index
fruits.forEach(function(fruit, index) {
  console.log(`${index}: ${fruit}`);
});

Exercice 3 : Additionner les éléments d'un tableau de nombres

Objectif : Utiliser forEach() pour calculer la somme des éléments d'un tableau de nombres.

Instructions : Crée un tableau nombres avec les éléments suivants : 5, 10, 15, 20. Utilise forEach() pour calculer et afficher la somme de ces nombres. 

const nombres = [5, 10, 15, 20];
let somme = 0;

// Utilise forEach pour additionner les nombres
nombres.forEach(function(nombre) {
  somme += nombre;
});

console.log("Somme totale : ", somme);

Exercice 4 : Modifier les éléments d'un tableau

Objectif : Utiliser forEach() pour modifier les éléments d'un tableau.

Instructions : Crée un tableau prix avec les éléments suivants : 100, 200, 300. Utilise forEach() pour appliquer une réduction de 10% sur chaque prix et affiche les nouveaux prix. 

const prix = [100, 200, 300];

// Utilise forEach pour réduire chaque prix de 10%
prix.forEach(function(p, index, array) {
  array[index] = p * 0.9;
});

console.log("Nouveaux prix : ", prix);

Exercice 5 : Filtrer les éléments d'un tableau (sans utiliser filter())

Objectif : Utiliser forEach() pour créer un nouveau tableau contenant uniquement les éléments qui répondent à une certaine condition.

Instructions : Crée un tableau âges avec les éléments suivants : 15, 18, 21, 17, 30. Utilise forEach() pour créer un nouveau tableau âges_adultes contenant uniquement les âges supérieurs ou égaux à 18 ans. 

const âges = [15, 18, 21, 17, 30];
const âges_adultes = [];

// Utilise forEach pour filtrer les âges
âges.forEach(function(âge) {
  if (âge >= 18) {
    âges_adultes.push(âge);
  }
});

console.log("Âges adultes : ", âges_adultes);
