Exercices pour comprendre la méthode some() en JavaScript

Dans cette leçon, tu vas apprendre à utiliser la méthode some() pour évaluer les éléments d'un tableau selon une condition spécifique. Cette méthode est utile pour vérifier si au moins un élément d'un tableau répond à un critère donné. Nous allons explorer cette méthode à travers une série d'exercices progressifs.

Exercice 1 : Vérifier la présence d'un nombre pair

Objectif : Utiliser some() pour vérifier si au moins un élément d'un tableau est un nombre pair.

Explication : La méthode some() prend une fonction de callback qui est exécutée sur chaque élément du tableau. Si cette fonction renvoie true pour au moins un élément, alors some() renvoie true, sinon elle renvoie false.

Instructions : Crée un tableau nombres avec les éléments suivants : 1, 3, 5, 7, 8. Utilise some() pour vérifier s'il y a au moins un nombre pair dans le tableau et affiche le résultat (true ou false). 

const nombres = [1, 3, 5, 7, 8];

// Utilise some pour vérifier la présence d'un nombre pair
const hasEvenNumber = nombres.some(function(nombre) {
  return nombre % 2 === 0;
});

console.log("Y a-t-il un nombre pair ? ", hasEvenNumber);

Exercice 2 : Vérifier si un élément dépasse une certaine valeur

Objectif : Utiliser some() pour vérifier si au moins un élément du tableau dépasse une valeur donnée.

Explication : Cet exercice t'aide à comprendre comment utiliser some() pour vérifier des conditions numériques. Ici, tu vérifieras si au moins un score dans un tableau dépasse une valeur spécifique.

Instructions : Crée un tableau scores avec les éléments suivants : 50, 60, 70, 80. Utilise some() pour vérifier s'il y a au moins un score supérieur à 75 et affiche le résultat (true ou false). 

const scores = [50, 60, 70, 80];

// Utilise some pour vérifier si un score dépasse 75
const hasHighScore = scores.some(function(score) {
  return score > 75;
});

console.log("Y a-t-il un score supérieur à 75 ? ", hasHighScore);

Exercice 3 : Vérifier la présence d'un mot spécifique dans un tableau

Objectif : Utiliser some() pour vérifier si un tableau de chaînes de caractères contient un mot spécifique.

Explication : Cet exercice montre comment some() peut être utilisé avec des chaînes de caractères pour vérifier la présence d'un élément particulier dans un tableau.

Instructions : Crée un tableau mots avec les éléments suivants : "chat", "chien", "poisson", "oiseau". Utilise some() pour vérifier si le mot "poisson" est présent dans le tableau et affiche le résultat (true ou false). 

const mots = ["chat", "chien", "poisson", "oiseau"];

// Utilise some pour vérifier si "poisson" est dans le tableau
const hasPoisson = mots.some(function(mot) {
  return mot === "poisson";
});

console.log("Le mot 'poisson' est-il présent ? ", hasPoisson);

Exercice 4 : Vérifier si au moins une personne est majeure

Objectif : Utiliser some() pour vérifier si au moins une personne d'un tableau est âgée de 18 ans ou plus.

Explication : Cet exercice démontre l'utilisation de some() dans le contexte de vérifications d'âge, ce qui est fréquent dans des applications de gestion de données personnelles.

Instructions : Crée un tableau âges avec les éléments suivants : 14, 16, 17, 19. Utilise some() pour vérifier s'il y a au moins une personne majeure dans le tableau et affiche le résultat (true ou false). 

const âges = [14, 16, 17, 19];

// Utilise some pour vérifier si au moins une personne est majeure
const hasAdult = âges.some(function(âge) {
  return âge >= 18;
});

console.log("Y a-t-il au moins une personne majeure ? ", hasAdult);

Exercice 5 : Vérifier si au moins un produit est en stock

Objectif : Utiliser some() pour vérifier si un tableau d'objets contient au moins un produit en stock.

Explication : Dans cet exercice, tu vas voir comment utiliser some() avec des objets pour vérifier une propriété spécifique, en l'occurrence si un produit est en stock.

Instructions : Crée un tableau produits avec des objets contenant deux propriétés : nom (chaîne de caractères) et enStock (booléen). Par exemple : 

const produits = [
  { nom: "Ordinateur", enStock: false },
  { nom: "Téléphone", enStock: false },
  { nom: "Tablette", enStock: true }
];

// Utilise some pour vérifier si au moins un produit est en stock
const hasProductInStock = produits.some(function(produit) {
  return produit.enStock === true;
});

console.log("Y a-t-il au moins un produit en stock ? ", hasProductInStock);
