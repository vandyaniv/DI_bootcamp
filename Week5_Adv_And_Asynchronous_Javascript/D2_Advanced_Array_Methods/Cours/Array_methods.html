Concepts Avancés JavaScript

Concept :
map() crée un nouveau tableau en appliquant une fonction à chaque élément du tableau d'origine.

Exemple :
Tu as un tableau de nombres et tu veux les multiplier par 2. 
const numbers = [1, 4, 9, 16];
const doubled = numbers.map(value => value * 2);
console.log(doubled); // [2, 8, 18, 32]
Explication :
  • Ancien tableau : [1, 4, 9, 16]
  • Nouvelle valeur : Chaque élément est multiplié par 2.
  • Nouveau tableau : [2, 8, 18, 32]
Question :
Imagine que tu as un tableau [10, 20, 30] et que tu veux ajouter 5 à chaque élément. Que retournerait map() ?

Concept :
filter() crée un nouveau tableau avec uniquement les éléments qui passent un test.

Exemple :
Tu as un tableau de mots et tu veux seulement ceux qui ont plus de 6 lettres. 
const words = ['spray', 'limit', 'elite', 'exuberant'];
const longWords = words.filter(word => word.length > 6);
console.log(longWords); // ['exuberant']
Explication :
  • Ancien tableau : ['spray', 'limit', 'elite', 'exuberant']
  • Condition : Longueur supérieure à 6 lettres.
  • Nouveau tableau : ['exuberant']
Question :
Si tu as un tableau [12, 5, 8, 130, 44] et que tu veux garder les nombres supérieurs à 10, quel serait le résultat ?

Concept :
reduce() combine tous les éléments d'un tableau en une seule valeur.

Exemple :
Tu as un tableau de nombres et tu veux calculer la somme. 
const array1 = [1, 2, 3, 4];
const sum = array1.reduce((acc, val) => acc + val, 0);
console.log(sum); // 10
Explication :
  • Ancien tableau : [1, 2, 3, 4]
  • Accumulation : On additionne chaque valeur à l'accumulateur.
  • Résultat : 10
Question :
Si tu veux multiplier tous les éléments d'un tableau [2, 3, 4], quel serait le résultat de reduce() ?

Concept :
La déstructuration permet d'extraire des valeurs d'un tableau ou d'un objet dans des variables distinctes.

Exemple :
Tu veux extraire les premiers éléments d'un tableau. 
const [a, b] = [10, 20];
console.log(a); // 10
console.log(b); // 20
Explication :
  • Ancien tableau : [10, 20]
  • Variables : a reçoit 10, b reçoit 20.
Question :
Que se passe-t-il si tu déstructures [30, 40, 50] en [x, y, ...rest] ?

Concept :
L'opérateur spread (...) permet d'étaler les éléments d'un tableau ou d'un objet.

Exemple :
Tu veux combiner plusieurs tableaux. 
const iterable = [10, 20, ...[30, 40, 50]];
console.log(iterable); // [10, 20, 30, 40, 50]
Explication :
  • Ancien tableau : [30, 40, 50] est éclaté.
  • Nouveau tableau : [10, 20, 30, 40, 50]
Question :
Si tu combines [1, 2] avec ...[3, 4], quel sera le résultat ?

Concept :
L'opérateur rest (...) regroupe les éléments restants d'un tableau ou d'un objet.

Exemple :
Tu veux capturer les éléments restants après les deux premiers. 
const [c, d, ...rest] = [10, 20, 30, 40, 50];
console.log(rest); // [30, 40, 50]
Explication :
  • Ancien tableau : [10, 20, 30, 40, 50]
  • Restant : [30, 40, 50]
Question :
Que se passe-t-il si tu déstructures [1, 2, 3, 4] avec [a, ...rest] ?

Concept :
En JavaScript, les tableaux sont des références. Pour cloner un tableau, tu peux utiliser le spread operator.

Exemple :
Tu veux faire une copie d'un tableau sans le modifier. 
const rainbow = ['red', 'orange', 'yellow'];
const rainbowClone = [...rainbow];
console.log(rainbowClone); // ['red', 'orange', 'yellow']
Explication :
  • Ancien tableau : ['red', 'orange', 'yellow']
  • Clone : Nouvelle copie indépendante.
Question :
Que se passe-t-il si tu modifies rainbow après avoir cloné avec ...rainbow ?
