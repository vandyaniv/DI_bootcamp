Exercices pour comprendre la méthode every() en JavaScript

Dans cette leçon, tu vas apprendre à utiliser la méthode every() pour vérifier si tous les éléments d'un tableau répondent à une condition spécifique. Ces exercices te guideront à travers des scénarios où cette méthode est particulièrement utile.

Exercice 1 : Vérifier si tous les nombres sont positifs

Objectif : Utiliser every() pour vérifier si tous les éléments d'un tableau de nombres sont positifs.

Explication : La méthode every() teste si tous les éléments d'un tableau passent un test mis en œuvre par une fonction fournie. Si la fonction renvoie true pour chaque élément, every() renvoie true, sinon false.

Instructions : Crée un tableau nombres avec les éléments suivants : 1, 3, 5, 7, 8. Utilise every() pour vérifier si tous les nombres sont positifs et affiche le résultat (true ou false). 

const nombres = [1, 3, 5, 7, 8];

// Utilise every pour vérifier si tous les nombres sont positifs
const allPositive = nombres.every(function(nombre) {
  return nombre > 0;
});

console.log("Tous les nombres sont-ils positifs ? ", allPositive);

Exercice 2 : Vérifier si tous les mots ont une certaine longueur

Objectif : Utiliser every() pour vérifier si tous les mots d'un tableau ont au moins 5 caractères.

Explication : Cet exercice montre comment every() peut être utilisé avec des chaînes de caractères pour vérifier une condition sur la longueur de chaque mot dans un tableau.

Instructions : Crée un tableau mots avec les éléments suivants : "arbre", "maison", "ciel", "soleil". Utilise every() pour vérifier si tous les mots ont au moins 5 caractères et affiche le résultat (true ou false). 

const mots = ["arbre", "maison", "ciel", "soleil"];

// Utilise every pour vérifier si tous les mots ont au moins 5 caractères
const allLongEnough = mots.every(function(mot) {
  return mot.length >= 5;
});

console.log("Tous les mots ont-ils au moins 5 caractères ? ", allLongEnough);

Exercice 3 : Vérifier si tous les âges sont au-dessus d'une certaine limite

Objectif : Utiliser every() pour vérifier si tous les âges d'un tableau sont supérieurs ou égaux à 18 ans.

Explication : Cet exercice démontre l'utilisation de every() dans le contexte de vérifications d'âge, un scénario courant pour des conditions d'éligibilité ou d'accès.

Instructions : Crée un tableau âges avec les éléments suivants : 18, 22, 19, 30. Utilise every() pour vérifier si tous les âges sont au moins 18 ans et affiche le résultat (true ou false). 

const âges = [18, 22, 19, 30];

// Utilise every pour vérifier si tous les âges sont supérieurs ou égaux à 18 ans
const allAdults = âges.every(function(âge) {
  return âge >= 18;
});

console.log("Tous les âges sont-ils supérieurs ou égaux à 18 ans ? ", allAdults);

Exercice 4 : Vérifier si tous les produits sont en stock

Objectif : Utiliser every() pour vérifier si tous les produits d'un tableau d'objets sont en stock.

Explication : Cet exercice montre comment every() peut être utilisé avec des objets pour vérifier une condition sur une propriété spécifique de chaque objet dans un tableau.

Instructions : Crée un tableau produits avec des objets contenant deux propriétés : nom (chaîne de caractères) et enStock (booléen). Par exemple : 

const produits = [
  { nom: "Ordinateur", enStock: true },
  { nom: "Téléphone", enStock: true },
  { nom: "Tablette", enStock: true }
];

// Utilise every pour vérifier si tous les produits sont en stock
const allInStock = produits.every(function(produit) {
  return produit.enStock === true;
});

console.log("Tous les produits sont-ils en stock ? ", allInStock);

Exercice 5 : Vérifier si tous les numéros de téléphone sont valides

Objectif : Utiliser every() pour vérifier si tous les numéros de téléphone d'un tableau respectent un format spécifique (par exemple, 10 chiffres).

Explication : Cet exercice met en pratique every() pour la validation de formats de données, ici des numéros de téléphone, un cas d'usage courant pour valider des entrées utilisateur.

Instructions : Crée un tableau téléphones avec les éléments suivants : "1234567890", "0987654321", "1122334455". Utilise every() pour vérifier si tous les numéros ont 10 chiffres et affiche le résultat (true ou false). 

const téléphones = ["1234567890", "0987654321", "1122334455"];

// Utilise every pour vérifier si tous les numéros de téléphone sont valides
const allValidPhones = téléphones.every(function(téléphone) {
  return téléphone.length === 10;
});

console.log("Tous les numéros de téléphone sont-ils valides ? ", allValidPhones);
