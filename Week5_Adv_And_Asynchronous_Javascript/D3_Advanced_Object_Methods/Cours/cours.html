Leçons sur les Objets et Classes en JavaScript

Imaginons : Tu as un classeur avec des onglets pour chaque catégorie, comme les noms, les âges, et les adresses. Les méthodes des objets te permettent de manipuler et d'organiser ces informations dans le classeur.

  • Object.keys() : Liste tous les onglets du classeur.

    • Analogie : Comme si tu écrivais une liste des onglets pour voir ce qui est dans le classeur.

    const person = { name: 'John', age: 30, city: 'Paris' };
console.log(Object.keys(person)); // ["name", "age", "city"]
  • Object.values() : Liste toutes les informations dans chaque onglet.

    • Analogie : Comme si tu listais toutes les informations sous chaque onglet.

    console.log(Object.values(person)); // ["John", 30, "Paris"]
  • Object.entries() : Combine les onglets et les informations sous forme de paires.

    • Analogie : Comme si tu écrivais chaque onglet avec son contenu côte à côte.

    console.log(Object.entries(person)); // [["name", "John"], ["age", 30], ["city", "Paris"]]
  • Object.fromEntries() : Crée un classeur à partir de paires d’onglets et d’informations.

    • Analogie : Comme si tu créais un nouveau classeur à partir d'une liste de paires onglet-information.

    const entries = [["name", "John"], ["age", 30], ["city", "Paris"]];
const newPerson = Object.fromEntries(entries);
console.log(newPerson); // { name: "John", age: 30, city: "Paris" }

Imaginons : Tu as un tiroir avec des boîtes, chacune contenant des informations spécifiques, comme un prénom, un nom de famille, et un âge. La déstructuration te permet de sortir ces informations des boîtes facilement.

  • Déstructuration Simple : Ouvrir une boîte et prendre ce dont tu as besoin.

    • Analogie : Comme si tu ouvrais une boîte étiquetée "adresse" et que tu prenais juste la rue et la ville.

    const address = { street: 'Main St', city: 'New York' };
const { street, city } = address;
console.log(street); // Main St
console.log(city); // New York
  • Déstructuration Imbriquée : Ouvrir une boîte qui contient une autre boîte et en extraire les informations.

    • Analogie : Comme si tu ouvrais une grande boîte, puis une plus petite à l'intérieur pour accéder aux informations.

    const student = { name: 'Jane', scores: { maths: 90, english: 80 } };
const { name, scores: { maths, english } } = student;
console.log(`${name} a eu ${maths} en maths et ${english} en anglais.`);
  • Déstructuration dans une Fonction : Utiliser la déstructuration directement dans les paramètres d'une fonction.

    • Analogie : Comme si tu demandais directement les éléments dont tu as besoin sans avoir à ouvrir chaque boîte toi-même.

    function showStudent({ name, scores: { maths, english } }) {
  console.log(`${name} a eu ${maths} en maths et ${english} en anglais.`);
}

Imaginons : Tu as un moule en forme de gâteau. Le clonage te permet de faire une copie exacte du gâteau sans toucher à l'original.

  • Clonage Superficiel : Utiliser un moule pour copier uniquement la surface du gâteau.

    • Analogie : Comme si tu faisais une copie de la forme extérieure du gâteau, mais que l'intérieur reste connecté à l'original.

    const car = { color: 'blue', details: { engine: 'V8' } };
const carClone = { ...car };
car.color = 'red';
console.log(carClone.color); // blue (car le clonage est superficiel)
  • Clonage Profond : Faire une copie complète, y compris les détails intérieurs du gâteau.

    • Analogie : Comme si tu créais une copie complète du gâteau, y compris les couches internes.

    const deepClone = JSON.parse(JSON.stringify(car));
car.details.engine = 'V6';
console.log(deepClone.details.engine); // V8 (car le clonage est profond)

Imaginons : Tu veux construire plusieurs maisons identiques. Pour cela, tu crées un plan. Ce plan décrit comment chaque maison doit être construite : combien de chambres, la taille de la cuisine, etc.

  • Classe = Plan : La classe est comme le plan de la maison. C'est un modèle pour créer des objets.
  • Maison = Objet : Chaque maison que tu construis à partir du plan est un objet.
class Maison {
  constructor(chambres, cuisine) {
    this.chambres = chambres;
    this.cuisine = cuisine;
  }
}

let maMaison = new Maison(3, 'grande');

Imaginons : Lorsque tu décides de construire une maison, tu passes des instructions spécifiques : combien de chambres, quelle taille de cuisine. Ces instructions sont utilisées pour créer chaque maison.

  • Construction = Constructeur : Le constructeur est la méthode spéciale utilisée pour construire l'objet (maison) à partir de la classe (plan).
class Maison {
  constructor(chambres, cuisine) {
    this.chambres = chambres;
    this.cuisine = cuisine;
  }
}

let maMaison = new Maison(3, 'grande');

Imaginons : Une fois ta maison construite, tu peux y faire différentes choses, comme ouvrir les fenêtres ou allumer les lumières. Ces actions sont comme des fonctionnalités de la maison.

  • Fonctionnalités = Méthodes : Les méthodes sont des actions que l'objet (maison) peut effectuer.
class Maison {
  constructor(chambres, cuisine) {
    this.chambres = chambres;
    this.cuisine = cuisine;
  }

  ouvrirFenetre() {
    console.log('Fenêtre ouverte');
  }
}

let maMaison = new Maison(3, 'grande');
maMaison.ouvrirFenetre(); // Fenêtre ouverte

Imaginons : Tu veux construire une maison spéciale qui a non seulement toutes les fonctionnalités de la maison de base, mais aussi un jardin. Plutôt que de créer un nouveau plan de zéro, tu prends le plan existant et tu ajoutes un jardin.

  • Maison avec Jardin = Sous-classe : Une sous-classe hérite des fonctionnalités de la classe parente et peut avoir des fonctionnalités supplémentaires.
class Maison {
  constructor(chambres, cuisine) {
    this.chambres = chambres;
    this.cuisine = cuisine;
  }

  ouvrirFenetre() {
    console.log('Fenêtre ouverte');
  }
}

class MaisonAvecJardin extends Maison {
  constructor(chambres, cuisine, jardin) {
    super(chambres, cuisine);
    this.jardin = jardin;
  }

  ouvrirPortail() {
    console.log('Portail ouvert');
  }
}

let maMaisonJardin = new MaisonAvecJardin(3, 'grande', 'grand');
maMaisonJardin.ouvrirFenetre(); // Fenêtre ouverte
maMaisonJardin.ouvrirPortail(); // Portail ouvert

Explications sur la Déstructuration en JavaScript

Explication :

La déstructuration permet d'extraire des valeurs d'un tableau et de les assigner à des variables distinctes.

Exemple :

const arr = ['🥓', '🍕', '🍟'];
        
        const [bacon, pizza, fries] = arr;
        
        console.log(pizza); // 🍕

Analogie :

Imaginez que vous avez une boîte contenant du bacon, une pizza, et des frites. Grâce à la déstructuration, vous pouvez facilement sortir chaque élément de la boîte et les placer sur des assiettes distinctes.

Explication :

La déstructuration permet également d'extraire des propriétés d'un objet imbriqué.

Exemple :

const obj = {
            parent: {
                child: '👶'
            }
        };
        
        const { parent: { child } } = obj;
        
        console.log(child); // 👶

Analogie :

Imaginez une boîte (objet parent) contenant une plus petite boîte (enfant). La déstructuration vous permet d'ouvrir directement la plus petite boîte pour accéder à son contenu sans avoir à manipuler directement la boîte parent.

Explication :

La déstructuration vous permet aussi de renommer les propriétés lorsque vous les extrayez d'un objet.

Exemple :

const obj = {
            shroom: '🍄',
            banana: '🍌',
            pepper: '🌶️'
        };
        
        const { shroom: mushroom } = obj;
        
        console.log(mushroom); // 🍄

Analogie :

Imaginez que vous avez un panier contenant un champignon, une banane, et un piment. Vous pouvez décider de renommer le champignon en 'mushroom' pour le rendre plus compréhensible dans un autre contexte, tout en conservant le lien avec l'élément original.
